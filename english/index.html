    A signature page was created to validate the reports from both the agent and the client.

    Technologies used at APRR:

    PowerApps

    PowerApps is a platform that allows the creation of custom applications without requiring programming skills. It is based on NoCode/LowCode using PowerFx, a formula language similar to Excel. It enables the creation of applications for mobile devices and web browsers.

    PowerAutomate

    Power Automate is a workflow service that automates tasks and processes between different applications and services. It allows the creation of automated workflows to synchronize files, collect data, send notifications, etc.

    SharePoint

    SharePoint facilitates collaborative and simple file sharing. It allows the creation of document libraries, lists, and websites to enhance team collaboration. It is often used by the Power Platform to store data and files associated with applications created using PowerApps and Power Automate.

    Dataverse

    Dataverse is a cloud database that allows storing and managing data for applications created with PowerApps and Power Automate. It offers advanced data management, security, and integration features with other Microsoft services. (This service is more expensive than SharePoint).
